# AgentKraft Site

Standalone marketing site for `AgentKraft`, extracted from the AGS workspace as a dedicated Vite + React frontend.

## Stack

- React 18
- Vite 5
- Vitest + Testing Library

## Run

```powershell
npm install
npm run dev
```

## Test

```powershell
npm test -- --run
npm run build
```

## What It Contains

- Animated hero section
- Product positioning for `AgentFabric`, `AGS Sovereign`, and `Gavel Runtime`
- Architecture section
- Future-vision section

This folder is structured so it can be moved into its own Git repository without changes.
